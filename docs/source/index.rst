************************
Open5Gs-UI Documentation
************************

Open5Gs-UI is an extension for `Open5Gs`_. Designed to scrape data from
5G-Network function, analyse network traffic and display scraped data via
web server.

.. _Open5Gs: https://open5gs.org/
